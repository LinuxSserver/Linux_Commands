
# cat Command

Read File, Write File And Make New File.


```
cat file_name > new_file/old_file
```

```
cat file_name >> new_file/old_file
```
```
cat <<EOF>> file_name
```

```
cat <<EOF>> new_file_name
```

```
cat file_name
```

```
cat -n file_name
```

```
cat --help
```

```
cat --version
```
