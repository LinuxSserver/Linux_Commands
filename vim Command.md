
# vim Command

Text Editor.

**vim/vi Mode**

-   i

    Insert

-   r

    Replace

-   v

    Visual

-   esc
    
    Command Mode

**vim/vi Mode Switch**

-   :w
    
    Save file

-   :q

    Quit file

-   :q!

    Exit file

-   :wq!

    Save And Exit

-   :set num

    Set Numbers  

```
vim file_name
```
